# Vice President, Global Manufacturing Strategy - Candidate Vision

An independent candidate campaign by Russell Dudek for a confidential global manufacturing leadership opportunity.

## Campaign thesis

**Change the shape. Hold the promise.**

The campaign argues that a complex global manufacturing network should be able to change capacity, footprint, technology posture, and operating configuration while preserving an explicit customer promise and one enterprise operating contract.
